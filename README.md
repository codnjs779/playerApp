# Title

유튜브Api 활용한 인기동영상, 검색기능 페이지

## Install

```
npx create-react-app [폴더명]
npm install -g yarn
yarn add axios
yarn add @fortawesome/fontawesome-free

```

## Usage

```
yarn start

```
